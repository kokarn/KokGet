KokGet
======
The goal of this flexget extension is to make it really really easy to add movies and series to the download queue.

Scenarios that should be really easy to do:
 * Download a movie now
 * Download a movie when it comes out in the right quality
 * Download the first episode of a tv-series
 * Download the first season of a tv-series
 * Add tv-series to auto-download list
 * Remove tv-series from auto-download list
 * See what's in the download queue and remove unwanted items
